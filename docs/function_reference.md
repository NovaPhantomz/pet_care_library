# Function Reference – Pet Care Library

## 1. validate_pet_name(name)
Checks if a pet name is valid and non-empty.

## 2. validate_pet_age(age)
Ensures age is numeric and positive.

## 3. validate_pet_weight(weight)
Validates pet weight to be a realistic number.

## 4. format_reminder_message(pet_name, task)
Creates a simple reminder message like “Reminder: feed Luna today!”

## 5. convert_minutes_to_hours(minutes)
Converts minutes into readable “hr/min” format.

## 6. calculate_food_portion(weight, activity_level)
Estimates a pet’s daily food portion based on activity.

## 7. calculate_next_vet_visit(last_visit_date)
Adds one year to a given date for next vet appointment.

## 8. calculate_walk_distance(duration, pace)
Calculates estimated walk distance in kilometers.

## 9. filter_pets_by_species(pet_list, species)
Filters a list of pet dictionaries by species.

## 10. generate_health_summary(pet_data)
Summarizes a pet’s health info including visits and weight.

## 11. log_care_event(pet_name, event_type, timestamp, notes)
Logs a care activity (feeding, walking, etc.) to a text file.

## 12. calculate_average_activity(pet_logs)
Computes average time spent in pet activities.

## 13. send_care_alerts(pet_data)
Simulates reminders or alerts for each pet.

## 14. export_pet_report(pet_data, output_path)
Creates a simple report file listing pet details.
